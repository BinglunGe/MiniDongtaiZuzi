怎样购买 WinRAR 和 RAR 许可证

如果您需要购买商业版本，您可以在 www.winrar.com.cn 查看最新的价格并在线订购。

提交了您的注册费以后，您会收到一封电子邮件包含您选择的用户名字符串对应的注册 key 文件。购买许可证时请提供一个有效的电子邮件地址，它用来发送给您册文件。

注册电子邮件也会包含必要的指导，只要参照执行。下面我们提供了典型注册手续的简要说明，但是在电子邮件中的指导比此文件更新，并应该有更高的优先级。

如果您使用 WinRAR，您应该把注册 key 文件（rarreg.key）复制到 WinRAR 文件夹或 %APPDATA%\WinRAR 文件夹中。默认的 WinRAR 文件夹是“C:\Program Files\WinRAR”，但是用户能在安装 WinRAR 时修改它。您也可以拖放 rarreg.key 文件到 WinRAR 窗口来注册。

如果这个 key 是 RAR 或 zip 压缩文件，请从压缩文件中解压 rarreg.key 来注册您的 WinRAR。如果压缩文件的名称是 rarkey.rar，安装 key 文件的另一种方法是在WinRAR 中打开此压缩文件，并在提示时选择“是”来确认。

如果您使用 RAR/Unix 和 RAR for macOS，您应该把 rarreg.key 复制到您的 home 目录或下列目录之一: /etc, /usr/lib, /usr/local/lib, /usr/local/etc。如果您愿意，您可以把它改名为 .rarreg.key 或 .rarregkey，但 rarreg.key 也是可以的。

现在，WinRAR、命令行 RAR for Unix 和 macOS 使用相同的注册 key 格式，所以您可以使用当前 WinRAR 和 RAR 版本的同一个注册码用于所有这些平台。如果 WinRAR 和 RAR 的版本与此发布版本号不同则无法保证上述方法。
